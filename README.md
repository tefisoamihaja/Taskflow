TaskFlow — Application To-Do List avec React & Vite

TaskFlow est une application web de gestion de tâches développée en React.js. 
Ce projet met en pratique les fondamentaux de React (états, propriétés, découpage en composants, approche UI-First).



I- Fonctionnalités :
* Gestion des tâches :
    - Ajout de tâches avec génération d'identifiants uniques (Create).
    - Marquage des tâches (Edit).
    - Suppression de tâches (Delete).

* Suivi dynamique :
    - Compteur de tâches réalisées et restantes.
    - Affichage conditionnel de messages d'encouragement lorsque toutes les tâches sont accomplies.
    - Textes dynamiques (compteurs, pluriels) 

* Architecture modulaire :
    - Approche UI-First pour le prototypage.
    - Styles isolés avec les CSS Modules.



II- Tech Stack & Outils
- Framework/Bibliothèque : React.js
- Tooling & Bundler : Vite (npm create vite@latest)
- Styling : CSS / CSS Modules
- Langage : JavaScript (ES6+)



III- Installation et Lancement: 
1- Cloner le dépôt :
- Bash
- git clone 
- cd taskflow


2- Installer les dépendances :
Bash
npm install


3- Lancer le serveur de développement :
Bash
npm run dev


4- Ouvrir dans le navigateur :
Rendez-vous sur http://localhost:5173.
